
# AI Interview Coach

AI Interview Coach is an AI-powered interview preparation platform designed to help students and professionals practice interviews more effectively.

The application supports multiple interview types including Technical, HR, Behavioral, System Design, and Mixed Interviews. Users can customize the role, experience level, difficulty level, answer style, and number of questions to create personalized interview sessions.

## Features

- AI-generated interview questions
- Mock interview simulation
- Technical, HR, Behavioral, and System Design interview modes
- Answer evaluation and scoring
- AI follow-up questions
- Progress tracking
- Performance dashboard
- Weak area detection
- Personalized study roadmap
- Interview history tracking
- Gemini API integration
- Ollama (Llama 3.2) support for local AI inference

## Tech Stack

- HTML
- CSS
- JavaScript
- Gemini API
- Ollama
- Llama 3.2

## Use Cases

- Interview preparation
- Skill assessment
- Technical practice
- HR interview practice
- Learning roadmap generation
- Self-evaluation and improvement
